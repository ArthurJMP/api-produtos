# TechFood Solutions - TCC 2026

## Descrição do Projeto
A TechFood Solutions é uma solução de autoatendimento digital voltada para micro e pequenos comércios do setor gastronômico. O foco central é a eliminação de filas no balcão e a isenção de taxas abusivas de aplicativos de delivery tradicionais. 

O sistema foi projetado como um **Progressive Web App (PWA)** acessível via QR Code, permitindo que o cliente realize pedidos e pagamentos via PIX diretamente pelo próprio smartphone, de forma instantânea e sem a necessidade de instalar aplicativos.
